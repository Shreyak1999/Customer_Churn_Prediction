# Customer Churn Prediction


**Overview** : This project aims to build a scalable pipeline that predicts the likelihood of churn based on customer demographics, account details, and usage behavior. 

**Goal of the Project**: Predict whether a customer is likely to churn (cancel subscription/stop using service).

**Dataset Used**: Telco Customer Churn Dataset | ***Link***: https://www.kaggle.com/datasets/blastchar/telco-customer-churn


## Tech Stack:

* Python 3.8+
* VS Code
* Pandas Matplotlib
* Streamlit


## Workflow

-> Data Ingestion 
-> Data Cleaning 
-> Feature Engineering 
-> Feature Assembly 
-> Model Training 
-> Model Evaluation 
-> Insights 
-> Deployment


## How to use the Repository

* Clone the Repo:
    * git clone https://github.com/Shreyak1999/Customer_Churn_Prediction.git
    * cd Customer_Churn_Prediction

* Install the dependencies
    * pip install -r requirements.txt

* Run as below:
    * eda
    * data_preprocessing
    * model
    * insights
    * pipeline
